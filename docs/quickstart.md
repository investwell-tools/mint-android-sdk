# Quick Start

Initialize Mint SDK in your Application class.

Kotlin:

```kotlin
Mint.initialize(
    context = applicationContext,
    apiKey = "YOUR_API_KEY"
)
```

Java :

```Java
Mint.initialize(
    getApplicationContext(),
    "YOUR_API_KEY"
);
```
