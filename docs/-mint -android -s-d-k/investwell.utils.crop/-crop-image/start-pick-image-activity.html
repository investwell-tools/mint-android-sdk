open fun startPickImageActivity(@NonNull activity: Activity)

Start an activity to get image for cropping using chooser intent that will have all the available applications for the device like camera (MyCamera), galery (Photos), store apps (Dropbox), etc. Use "pick_image_intent_chooser_title" string resource to override pick chooser title.
